House Price Predictor (Gradio + Random Forest)

A complete Machine Learning web app that predicts house prices using the California Housing Dataset.
This project features a beautiful Black & Orange themed UI built with Gradio and a Random Forest model for accurate predictions.

🚀 Features

🧠 Trained using RandomForestRegressor

📊 Uses California Housing Dataset (Scikit-learn)

⚡ Real-time predictions with Gradio UI

🎨 Custom Black & Orange themed interface

🌐 Shareable public link (Gradio share=True)

📍 Takes 8 real-world housing features

🛠️ Tech Stack

Python

Scikit-learn

Pandas & NumPy

Gradio
